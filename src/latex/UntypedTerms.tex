\text {The Set of untyped Lambda terms:} \\

\text {Variable - if } x \in V \text { then } x \in \Lambda \\
\text {Application - if } M,N \in \Lambda then (MN) \in \Lambda
\text {Abstraction - if } x \in V and M \in \lambda then (\lambda x . M) \in \Lambda

\text {Application is left associative, whereas Abstraction is right associative.  As well as this, application takes precedent over abstraction}
